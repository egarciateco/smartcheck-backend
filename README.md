# 🛒 SmartCheck API

API de comparación de precios de supermercados en Argentina. Desarrollada con FastAPI y desplegada en Render.

## 🚀 Características

- 🔍 Comparación de precios en tiempo real
- 📍 Filtrado por provincia y localidad
- 🏪 Múltiples comercios: Carrefour, Coto, Jumbo, Vital
- 💰 Precios basados en mercado argentino actualizado
- 📱 API RESTful compatible con móviles
